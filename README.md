# MU Annual Commuting Mileage Estimation  
**Miami University, Oxford, Ohio**

## Project Overview
This project estimates annual commuting mileage and associated CO₂ emissions for students, faculty, and staff at Miami University.  
The analysis was completed in support of the university’s **FY25 Climate Action Plan**, with the goal of providing a reproducible method for estimating Scope 3 transportation emissions and creating a baseline for future sustainability reporting.

## Project Objectives
- Estimate total annual commuting distance for students, faculty, and staff  
- Analyze average travel distances using survey responses and ZIP code data  
- Approximate yearly carbon emissions based on commuting frequency  
- Develop a reproducible workflow for sustainability reporting related to FY25  

## Data and Methods

### Data sources
- Miami University commuting survey data  
- ZIP code distance tables  
- Population counts for students, faculty, and staff  

### Software and tools
- R  
- R Markdown  
- tidyverse  
- dplyr  
- ggplot2  
- readxl  
- lubridate  

### Workflow summary
- Cleaned and organized survey responses by commuter category  
- Matched ZIP codes to estimated travel distances  
- Computed one-way and round-trip commute mileage  
- Scaled sample results to represent the full university population  
- Estimated yearly commuting mileage using average travel frequency  
- Generated tables and visualizations  
- Produced a reproducible HTML report using R Markdown  

## Key Results
Faculty and staff account for a large portion of total commuting mileage due to regular daily travel.  
Student commuting distances show greater variation, mainly influenced by off-campus housing locations.  
The results provide a baseline for tracking transportation-related emissions as part of the **FY25 Climate Action Plan**.
